# Gazfond Sensey Email V2: инструкция

## Что загрузить в Сенсей

- HTML: `gazfond-sensey-email.html`.
- Изображения: фон письма и три иконки из `assets/`.
- CTA-ссылка: `https://crm.gazfond-pn.ru/landing?name=ZevLanding`.
- Тема: `Слишком молода для скучной пенсии`.
- Preheader: `Будущее не обязано быть серым. ПДС помогает собрать понятный и гибкий сценарий будущего.`

## Placeholders

- `{{IMAGE_1_URL}}` — фон письма.
- `{{IMAGE_2_URL}}` — иконка блока `Ваши правила`.
- `{{IMAGE_3_URL}}` — иконка блока `Не копить тревогу. Копить опору`.
- `{{IMAGE_4_URL}}` — иконка блока `Выберите себя заранее`.

## Перед отправкой

- Заменить все image placeholders на реальные CDN URL Сенсея.
- Проверить обе CTA-ссылки.
- Проверить мобильные ширины 320, 375 и 414 px.
- Проверить Gmail, Mail.ru, Яндекс.Почту, Apple Mail и Outlook.
- Проверить alt-тексты и юридический футер.
- Убедиться, что нет обещаний доходности.
- Проверить UTM-метки, если они будут добавлены к CTA.

## Допущение

Для GitHub Pages использована web-optimized JPG-версия фона, чтобы публикация прошла через API. Локальный финальный пакет содержит более качественный PNG-фон.
